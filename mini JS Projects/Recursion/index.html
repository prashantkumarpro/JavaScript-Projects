What is Recursion?

Recursion means a function calling itself until it solves a smaller, simpler problem. Imagine you're asking someone for help, and they keep asking others until they reach someone who knows the answer. Once the answer is found, they all pass it back up to you.

Steps for Recursion:

  1. Base Case: The simplest situation where we stop asking for help (the function stops calling itself).
  2. Recursive Case: The case where the function calls itself again with a simpler version of the problem.

Example: Countdown

Let's say we want to count down from a number to 1.


function countdown(n) {
  // Base case: stop when n is 0
  if (n === 0) {
    console.log("Done!");
    return;
  }
  // Recursive case: keep calling the function
  console.log(n);
  countdown(n - 1); // Call the function with n - 1
}

countdown(5);

How It Works:

Base Case: If n is 0, the function stops and prints "Done!"

Recursive Case: If n is greater than 0, it prints the number and then calls itself with n - 1.

When you call countdown(5), it works like this:

  • countdown(5) prints 5 and calls countdown(4)
  • countdown(4) prints 4 and calls countdown(3)
  • countdown(3) prints 3 and calls countdown(2)
  • countdown(2) prints 2 and calls countdown(1)
  • countdown(1) prints 1 and calls countdown(0)
  • countdown(0) hits the base case and prints "Done!"

Recap:

  • Recursion means a function keeps calling itself until it reaches a base case.
  • Base Case is the condition where the recursion stops.
  • Recursive Case is where the function keeps calling itself with a smaller or simpler problem.
